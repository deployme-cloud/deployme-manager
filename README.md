# deployme-manager

Espace client Deployme.cloud — Dashboard clusters, monitoring, billing.

**Stack** : SvelteKit + TailwindCSS 4 + Zitadel (OIDC)
**Deploy** : Netlify -> `app.deployme.cloud`

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Environment

Copy `.env.example` to `.env` and fill in the values.
